# GMOコイン API 仕様（参照用）

公式: https://api.coin.z.com/docs/

## エンドポイント

- 本番: `https://api.coin.z.com`
- 認証: API キーとシークレット（環境変数から読み込む想定）

## 主な制約

| 項目 | 内容 |
|------|------|
| 最小注文単位 | 0.00001 BTC（通貨ペアにより異なる。XRP 等は別途要確認） |
| レート制限 | API ごとに制限あり。連続呼び出し時は間隔・リトライを考慮する |
| エラーレスポンス | HTTP ステータスと JSON でエラー内容を返す。必ず捕捉しログに残す |

## エラーハンドリング

- 4xx / 5xx や API 側のエラーコードは捕捉し、ログに残すこと。
- 連続エラー時は `risk_management.max_consecutive_errors` に従い停止することを推奨（config 参照）。

## 取引手数料（参考）

- Maker -0.01%, Taker 0.05%（公式ドキュメントで最新を確認すること）
